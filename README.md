# Admin Dashboard – The Odin Project

A comprehensive, responsive admin interface built as part of **The Odin Project's** JavaScript Path. This project focuses on advanced CSS Grid layouts, Flexbox alignment, and creating a cohesive UI/UX using SVG icons.

## 🚀 Live Demo
(https://ramicoder.github.io/Admin-dashboard/)

## 🛠 Features
* **Complex Grid Layout:** Utilizes CSS Grid for the primary three-tier dashboard structure (Sidebar, Navbar, and Content area).
* **Dynamic Scaling:** Implements `clamp()` functions for typography and spacing to ensure a fluid experience across different screen sizes.
* **SVG Integration:** Custom-styled Material Design Icons used throughout the sidebar and action menus for a crisp, professional look.
* **Themed Components:** Includes custom-styled cards for "Partners," "Announcements," and "Trending" leagues.

## 💻 Tech Stack
* **HTML5:** Semantic structure.
* **CSS3:** Custom properties, CSS Grid, Flexbox, and responsive design.
* **Typography:** [Sentient](https://www.fontshare.com/fonts/sentient) by Fontshare.

## 🧠 Key Learnings

* **Grid Mastery:** Managing `grid-template-columns` and `grid-template-rows` to align diverse content types.
* **Sizing Logic:** Learning when to use `fr` units versus `%` to maintain layout integrity.
* **UI/UX:** Balancing information density while maintaining readability using white space and card-based design.
